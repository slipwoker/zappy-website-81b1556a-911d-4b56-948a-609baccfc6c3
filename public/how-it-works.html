Algebra

Solving quadratic equations with step-by-step guidance

Given: x² + 5x + 6 = 0
Step 1: Factor the quadratic
(x + 2)(x + 3) = 0
Step 2: Apply zero product property
x + 2 = 0 or x + 3 = 0
Solution: x = -2 or x = -3

We factored the expression and found two solutions by setting each factor equal to zero.

Geometry

Finding angles and understanding geometric relationships

Given: Triangle ABC, ∠A = 45°, ∠B = 60°
Find: ∠C
Step 1: Sum of angles in triangle = 180°
∠A + ∠B + ∠C = 180°
Step 2: Substitute known values
45° + 60° + ∠C = 180°
Solution: ∠C = 75°

We used the fundamental property that all angles in a triangle sum to 180 degrees.

Word Problem

Translating real-world scenarios into mathematical equations

Problem: Sarah has 3 times as many books
as Tom. Together they have 48 books.
How many books does each have?
Step 1: Let x = Tom's books
Then 3x = Sarah's books
Step 2: x + 3x = 48
4x = 48
Solution: Tom has 12, Sarah has 36

We defined a variable for the unknown quantity and set up an equation based on the relationship described.
